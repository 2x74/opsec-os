OpSec OS is the only distro based on Void Linux.
This distro already comes with the nonfree and mulitlib repos, along with flatpak (and flathub) 
For extra repos (like ones in github), please look below at the OTHER REPOS section

═══════════════════════════════════════════════════════════════════
INSTALLATION
═══════════════════════════════════════════════════════════════════
To install OpSec OS, open a terminal and type:

    sudo apt install opsec

PACKAGE MANAGEMENT
═══════════════════════════════════════════════════════════════════
OpSec OS uses the XBPS package manager:

    xbps-install <package>      - Install a package
    xbps-remove <package>       - Remove a package
    xbps-query -Rs <keyword>    - Search for packages
    xbps-install -Su            - Update system
